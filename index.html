Representative Projects & Case Studies

Motion Planning for Ackermann Vehicles using Dubins-Constrained RRT Variants

Class of problem: Motion planning, nonholonomic constraints, sampling-based planning

Developed and evaluated Dubins-constrained motion planners (RRT, Bi-RRT, P-RRT*) for Ackermann-steered vehicles across 300 simulation trials. Analyzed trade-offs between feasibility, planning time, and computational overhead in nonholonomic planning.

Identified BiRRT*-Dubins as the optimal planner, achieving a 10√ó speedup (0.043s average runtime) with a 100% success rate, while quantifying a 292% computational overhead relative to holonomic planning.

Key learnings: Constraint-aware planning, empirical evaluation at scale, and performance trade-offs in real-time planners.

Learning from Long-Tail Driving Scenarios in Simulation

Class of problem: Simulation-based learning, reinforcement / imitation learning, rare-event analysis

Building an end-to-end pipeline using the CARLA simulator to surface rare and safety-critical driving scenarios. The system focuses on identifying long-tail events and improving agent behavior through reinforcement and imitation learning.

Emphasis is placed on data collection, scenario mining, evaluation workflows, and policy improvement rather than purely algorithmic novelty.

Key learnings: Long-tail data challenges, simulation realism limits, and evaluation-driven autonomy development.

Real-Time Kinematics and Trajectory Control for UR5 6-DOF Robot

Class of problem: Robot kinematics, control, real-time systems

Implemented a complete kinematic and control framework for a UR5 6-DOF manipulator. Replaced an unreliable iterative inverse kinematics solver with a differential velocity-level controller to enable stable, real-time trajectory execution.

Successfully executed circular, square, and sinusoidal trajectories with sub-3mm tracking accuracy, supported by interactive visualization of joint states and end-effector pose.

Key learnings: Numerical stability, control-loop design, and real-time execution constraints.

Production ML Pipelines for Anomaly Detection & Forecasting (Industry)

Class of problem: Distributed systems, streaming ML, observability

Designed and operated large-scale ML and data pipelines for unsupervised anomaly detection and time-series forecasting over high-volume telemetry data. Implemented alert clustering and correlation techniques that reduced alert noise by 20% and improved incident triage and MTTR.

Optimized microservices-based ingestion pipelines, reducing data ingestion latency by 20% and enabling faster real-time analysis. Integrated monitoring and observability pipelines using Prometheus and Grafana for production diagnostics.

Key learnings: Reliability at scale, observability-driven debugging, and ML system lifecycle management.
